# Implementação da whatsapp-web.js
Uma WhatsApp API client que se conecta através do WhatsApp Web browser

Utiliza Puppeteer para rodar uma instância real do Whatsapp Web, que evita bloqueio.

## Quick Links

* [Meu Linktree](https://linktr.ee/felpsgk) 

## Installation

O módulo está disponível no npm! `npm i whatsapp-web.js`

Necessário Node V12 mínimo.
